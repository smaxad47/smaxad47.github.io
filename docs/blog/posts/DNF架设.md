# DNF架设教程记录
记录架设贴吧:DNF台服中狗哥的神迹架设过程  
本次架设使用的是[XanderYe](https://github.com/XanderYe/dnf)大佬的DOCKER镜像进行部署,还有更为方便的[1995chen](https://github.com/1995chen/dnf),可惜不支持DP2插件,所以选择前者.
